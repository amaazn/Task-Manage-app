TaskFlow is a high-performance, full-stack application built for seamless task organization. It features a modern "Midnight Neon" UI, robust JWT authentication, and a cloud-synced dashboard.

🚀 **[View Live Project](https://task-manage-app-frontend.onrender.com)**

---

## 🌟 Key Features

- 🔐 **Secure JWT Auth**: Full registration and login flow with protected frontend/backend routes.
- 📊 **Intelligent Dashboard**: Real-time task metrics, completion percentages, and status tracking.
- 🛠️ **Full CRUD Logic**: Sophisticated task creation, editing, and deletion with instant UI updates.
- ✅ **Schema Validation**: Server-side and client-side data integrity powered by **Zod**.
- 🎨 **Modern UI/UX**: Clean, responsive layout designed with Tailwind CSS for optimal user experience across all devices.
- 🚀 **Cloud Architecture**: Deployed with a separate Frontend/Backend architecture on Render with a managed PostgreSQL database.

## 🛠️ Technical Stack

### Frontend
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Notifications**: React Hot Toast
- **HTTP Client**: Axios (with custom interceptors)

### Backend
- **Runtime**: Node.js & Express.js
- **ORM**: Prisma (Type-safe database client)
- **Database**: PostgreSQL
- **Security**: JWT, Cookie-parser, and CORS management
